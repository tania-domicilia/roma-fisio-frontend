# Roma Fisio Frontend

## Setup

1. Installa le dipendenze:

```bash
npm install
```

2. Avvia l'app:

```bash
npm run dev
```

## Struttura

- `src/pages`: Le pagine principali (login, dashboard, ecc.)
- `src/components`: Componenti riutilizzabili
- `src/services`: Chiamate API
- `src/routes`: Gestione del routing
